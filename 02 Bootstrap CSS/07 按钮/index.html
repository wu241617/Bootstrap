Bootstrap 按钮

任何带有 class .btn 的元素都会继承圆角灰色按钮的默认外观。但是 Bootstrap 提供了一些选项来定义按钮的样式
  1. .btn 为按钮添加基本样式
  2. .btn-default 默认/标准按钮
  3. .btn-primary 原始按钮
  4. .btn-success 表示成功的动作
  5. .btn-info 该样式可用于要弹出信息的按钮
  6. .btn-warning 表示需要谨慎操作的按钮
  7. .btn-danger 表示一个危险动作的按钮操作
  8. .btn-link 让按钮看起来像个链接 (仍然保留按钮行为)
  9. .btn-lg 制作一个大按钮
  10. .btn-sm 制作一个小按钮
  11. .btn-xs 制作一个超小按钮
  12. .btn-block 块级按钮(拉伸至父元素100%的宽度)
  13. .active 按钮被点击
  14. .disabled 禁用按钮

按钮大小

1,.btn-lg 这会让按钮看起来比较大。
2,.btn-sm 这会让按钮看起来比较小。
3,.btn-xs 这会让按钮看起来特别小。
4,.btn-block 这会创建块级的按钮，会横跨父元素的全部宽度。

按钮状态

Bootstrap 提供了激活、禁用等按钮状态的 class
1,激活状态 .active-->按钮在激活时将呈现为被按压的外观（深色的背景、深色的边框、阴影）。
按钮元素 添加 .active class 来显示它是激活的。 锚元素 添加 .active class 到 a 按钮来显示它是激活的。

2,禁用状态 .disabled-->当您禁用一个按钮时，它的颜色会变淡 50%，并失去渐变。
按钮元素 添加 disabled 属性 到 button 按钮。 锚元素 添加 disabled class 到 a 按钮。
注意：该 class 只会改变 a 的外观，不会改变它的功能。在这里，您需要使用自定义的 JavaScript 来禁用链接。

按钮标签

您可以在 a、button 或 input 元素上使用按钮 class。 但是建议您在 button 元素上使用按钮 class，避免跨浏览器的不一致性问题。
