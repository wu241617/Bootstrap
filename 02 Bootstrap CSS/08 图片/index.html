Bootstrap 图片

Bootstrap 提供了三个可对图片应用简单样式的 class：
1,.img-rounded：添加 border-radius:6px 来获得图片圆角。
2,.img-circle：添加 border-radius:500px 来让整个图片变成圆形。
3,.img-thumbnail：添加一些内边距（padding）和一个灰色的边框。

响应式图片
