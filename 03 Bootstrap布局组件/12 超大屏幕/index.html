Bootstrap 超大屏幕（Jumbotron）

本章将讲解 Bootstrap 支持的另一个特性，超大屏幕（Jumbotron）。顾名思义该组件可以增加标题的大小， 并为登陆页面内容添加更多的外边距（margin）。使用超大屏幕（Jumbotron）的步骤如下： 创建一个带有 class .jumbotron. 的容器 div>。 除了更大的 h1>，字体粗细 font-weight 被减为 200px。
