Bootstrap 按钮下拉菜单

使用 Bootstrap class 向按钮添加下拉菜单。如需向按钮添加下拉菜单，只需要简单地在在一个 .btn-group 中放置按钮和下拉菜单即可。您也可以使用 span class="caret"> /span> 来指示按钮作为下拉菜单。

分割的按钮下拉菜单

分割的按钮下拉菜单使用与下拉菜单按钮大致相同的样式，但是对下拉菜单添加了原始的功能。 分割按钮的左边是原始的功能，右边是显示下拉菜单的切换。
