Bootstrap 徽章（Badges）

本章将讲解 Bootstrap 徽章（Badges）。徽章与标签相似，主要的区别在于徽章的边角更加圆滑。 徽章（Badges）主要用于突出显示新的或未读的项。如需使用徽章，只需要把 span class="badge"> 添加到链接、Bootstrap 导航等这些元素上即可。 当没有新的或未读的项时，通过 CSS 的 :empty 选择器，徽章会折叠起来，表示里边没有内容。
